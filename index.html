Projects

Some things I have worked on.


Movie for Me

Nov 2016 - Present

This is a web application I built originally for my CS50 final project. It's purpose is sorting movies on Netflix. The back-end is in Python (Flask) and the front-end uses JavaScript as well as HTML/CSS. I am currently working on scraping streaming sites and using SQL to store and query movie data.

NBA Schedule Optimizer

Nov, Dec 2017

This was a final project for CS182, Artificial Intelligence. I worked on this with Jack Connolly, and we created a way to generate and optimize random, valid NBA schedules by limiting the number of back-to-back games and travel distance. Our method of optimization was using local search, specifically hill climbing and simulated annealing algorithms. This project was done entirely in Python.

MiniML

Apr, May 2017

This was the implementation of a subset of a function programming language using OCaml. It was my final project for CS51. I implemented both a dynamically scoped environment model and a subsitution model for the subset of the language. Unfortunately, this repo is private on GitHub because most students choose to do a very similar project for the class, but if you reach out I would be happy to talk more about it.
